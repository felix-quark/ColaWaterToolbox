.. py:function:: process(attr: Optional[Union[str, int]], csv: bool = False) -> str

   Returns a human readable string representation of a nullable field value.

   Turns a ``None`` value into ``"<Null>"`` and optionally strips out whitespace,
   commas, and quotation marks.

   :param attr: The attribute value to be processed.
   :type attr: Optional[str]
   :param csv: Whether to apply CSV pre-processing.
   :type csv: bool

   :returns: The processed attribute value.
   :rtype: str
