   Utilities for handling attribute values.

   .. rubric:: Examples

   .. code-block:: python

       import colawater.attribute as attr

       foo = None
       foo = attr.process(foo) # Returns "<Null>".

.. py:function:: process(attr: Optional[Any], csv: bool = False) -> str

   Returns a human readable string representation of a nullable field value.

   Turns a ``None`` value into ``"<Null>"`` and optionally strips leading and trailing whitespace,
   commas, and quotation marks for ease of use as a CSV.

   :param attr: The attribute value to be processed.
   :type attr: Optional[Union[Any]]
   :param csv: Whether to apply CSV pre-processing.
   :type csv: bool

   :returns: The processed attribute value.
   :rtype: str
