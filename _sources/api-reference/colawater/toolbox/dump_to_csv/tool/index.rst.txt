.. py:function:: execute(parameters: list[arcpy.Parameter]) -> None

   Entry point for Dump to CSV.

   Dumps full attribute tables of input layers to CSV files in the target directory.

   :param parameters: The list of parameters.
   :type parameters: list[arcpy.Parameter]

   Returns the parameters for Dump to CSV.

   :rtype: list[arcpy.Parameter]


.. py:function:: dump_to_csv(input_layers: List[Any], dest_dir: pathlib.Path) -> List[pathlib.Path]

   Dump attribute table of layers in input_layers to CSVs in dest_dir.

   :param input_layers: List of input items supported by Export Table.
                        (These types are private in ArcPy or hard-to-find, hence the ``Any`` annotation)
   :type input_layers: List[Any]
   :param dest_dir: Destination directory for exported files.
   :type dest_dir: Path

   :returns: List of paths to the exported files.
   :rtype: List[Path]
