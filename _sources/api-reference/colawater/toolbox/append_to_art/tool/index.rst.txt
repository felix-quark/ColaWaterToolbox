.. py:function:: append_to_art(wm_lyr: arcpy._mp.Layer, wm_where_clause: str, art_table: arcpy._mp.Table) -> list[tuple[Optional[str], Ellipsis]]

   Appends recent integrated and well-sourced mains from a given editor to the
   Asset Reference Table.

   :param wm_lyr: A water main layer from which mains will be read.
   :type wm_lyr: arcpy._mp.Layer
   :param wm_where_clause: A SQL where clause used to select a subset of wm_lyr.
   :type wm_where_clause: str
   :param art_table: The asset reference table.
   :type art_table: arcpy._mp.Table

   :returns: A list of records of appended mains.
   :rtype: list[tuple[Optional[str], ...]]

   :raises ExecuteError: An error ocurred in the tool execution.

   .. note:: Modifies input table.
